## 1. 产品概述

面向公关团队晨会使用的桌面端竞品声量简报工具，聚焦每日晨会汇报场景，快速整理昨日品牌与竞品舆情数据，支持简报编辑与投屏导出。

- 目标用户：品牌公关团队、舆情值班人员、外部代理公司
- 核心价值：提升晨会效率，将复杂舆情数据转化为清晰可汇报的简报内容
- 适用场景：每日品牌晨会、竞品舆情监控汇报、危机预警快速通报

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心诉求 |
|------|----------|----------|
| 公关专员 | 晨会前准备简报 | 快速生成数据、编辑重点内容 |
| 品牌负责人 | 晨会主持汇报 | 清晰的数据展示、重点突出 |
| 舆情值班员 | 日常监控与标注 | 异常峰值识别、风险标记 |
| 代理公司 | 协同汇报 | 多品牌组切换、统一简报格式 |

### 2.2 功能模块

1. **品牌组选择**：启动后选择所属品牌组，自动加载对应数据
2. **昨日声量概览看板**：提及量排名、正负面占比、传播平台分布
3. **晨会卡片编辑区**：拖拽异常峰值进入简报、补充判断说明
4. **导出预览窗口**：四栏布局（品牌/竞品/风险/机会）单页简报

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 品牌选择页 | 品牌组列表 | 展示可选择的品牌组卡片，支持搜索和快速切换 |
| 主工作台 | 声量概览看板 | 昨日提及量排名柱状图、正负面占比环形图、平台分布热力图 |
| 主工作台 | 异常峰值列表 | 当日/昨日异常声量峰值事件，可拖拽 |
| 主工作台 | 简报编辑区 | 接收拖拽卡片，支持文字编辑和排序 |
| 导出预览 | 四栏简报 | 品牌、竞品、风险、机会四栏布局，适合投屏展示 |

## 3. 核心流程

用户打开客户端 → 选择所属品牌组 → 自动加载昨日声量数据概览 → 浏览异常峰值列表 → 将重点事件拖拽至简报区 → 补充判断说明 → 调整卡片顺序 → 打开导出预览 → 确认后导出/投屏展示

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃藏青（#1a2744）搭配香槟金（#c9a962），专业稳重
- **辅助色**：正面对比绿（#2d9a5e）、负面警示红（#d94c4c）、中性灰（#6b7280）
- **背景**：深浅分层，左侧导航深色，主内容区浅灰底配白色卡片
- **字体**：标题使用思源宋体/Noto Serif SC 彰显专业质感，正文使用 Inter/思源黑体
- **卡片风格**：微圆角（4px）、精致投影、细边框，强调数据的正式感
- **图标风格**：线性图标，统一 1.5px 描边，专业简洁
- **整体调性**：财经杂志风 / 专业咨询报告感，克制而有品质

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|----------|---------|
| 品牌选择页 | 品牌组卡片 | 品牌 logo 占位、品牌名称、数据更新时间、悬浮光效 |
| 主工作台 | 顶部标题栏 | 品牌组名称、昨日日期、导出按钮、切换品牌 |
| 主工作台 | 声量排名区 | 横向柱状图，6 个品牌排名，自家品牌高亮 |
| 主工作台 | 情感分析区 | 环形图展示正负面中立占比，6 品牌可切换 |
| 主工作台 | 平台分布区 | 微博/微信/抖音/小红书/新闻/论坛 六平台分布 |
| 主工作台 | 异常峰值列表 | 时间线布局，可拖拽卡片，异常等级标签 |
| 主工作台 | 简报编辑区 | 四象限或列表布局，已选卡片，可编辑文案 |
| 导出预览 | 单页简报 | 四栏布局，大字号，高对比度，适合投屏 |

### 4.3 响应性

- 桌面端优先设计，最小支持 1280px 宽度
- 主内容区采用固定宽度 + 弹性布局
- 导出预览页支持全屏模式，适配投影仪分辨率
- 拖拽操作桌面端优化，不依赖触控

### 4.4 动效与交互

- 页面加载：卡片依次淡入，数据图表渐进式呈现
- 拖拽交互：拿起时轻微放大+阴影加深，放置区有吸附反馈
- 数据切换：数字滚动动画，图表缓动过渡
- 按钮交互：点击时微缩反馈，悬停时边框发光
